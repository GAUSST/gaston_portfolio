# Gaoussou Diarra Portfolio

This repository contains Gaoussou Diarra's personal developer portfolio. It presents web, mobile, full-stack, e-commerce, and AI-assisted workflow projects for both recruiters and freelance clients.

## Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES modules)
- Font Awesome
- Formspree for the contact form

## Main Sections

- Hero / positioning
- Services
- Packages
- Monthly Technical Support
- Partners
- Projects
- Skills
- Experience
- Contact

## Run Locally

Because the site uses JavaScript modules, serve it with a local static server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploy

This is a static site. Deploy the repository to any static host such as:

- Netlify
- Vercel
- GitHub Pages

## Notes

- English and French content is managed in `js/i18n.js`.
- Project card content is managed in `js/projects-data.js`.
- Theme toggle, language toggle, filtering, and contact form behavior are handled in `js/`.
